File Structure

The template is organized in a clean and logical directory structure for easy maintenance and scalability.

  • / root
    • css/ (All stylesheet files including main.css)
    • js/ (JavaScript logic for animations and sliders)
    • images/ (All theme images and graphic assets)
    • services/ (Detailed sub-pages for specific services)
    • Theme-Preview/ (High-resolution full-page screenshots)
    • Documentation/ (Setup and maintenance guides)
    • index.html (Main landing page)
    • about.html (Company information)
    • services.html (Overview of all services)
    • blog.html (Article listing page)
    • contact.html (Booking and inquiry page)
    • privacy.html (Data security policy)
    • terms.html (Service terms & conditions)

Image Change

All images used in the template are located in the images/ folder. To replace an image, simply overwrite the existing file or change the src attribute in the HTML.

Standard Image Roles:

  • logo.jpg: Your company logo.
  • hero.png: Main visual on the home page.
  • service_*.png: Icons/Images for service cards.
Dimension Recommendation: For hero images, use 1920x1080px. For service icons, 500x500px works best.

Contact Form Setup

The contact forms are configured to work seamlessly with FormSubmit. To start receiving emails, follow these steps:

  1. Open contact.html or index.html.
  2. Find the <form> element.
  3. Update the action URL with your verified email address.
<form action="https://formsubmit.co/your-email@domain.com" method="POST"> <!-- form fields --> </form>

Make sure to confirm the first email sent from FormSubmit to activate your inbox.
